## 📊 技术实现

### 前端技术栈
- **框架：** Vue 3 (Composition API)
- **路由：** Vue Router 4
- **HTTP：** Axios
- **构建：** Vite
- **页面数：** 7 个

### 后端技术栈
- **运行时：** Node.js
- **框架：** Express 5
- **数据库：** Supabase (PostgreSQL)
- **环境：** dotenv

### 数据库设计
- **表名：** orders
- **字段数：** 17 个
- **索引：** 3 个
- **约束：** CHECK 约束、NOT NULL 约束
